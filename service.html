様々な支援サービスを通じて、皆様の暮らしをサポートします

CORE CONSULTATION SUPPORT

基幹相談支援センター

草津市における相談支援の中核的な役割を担い、総合的・専門的な相談支援を実施します。 また、地域の相談支援体制の強化を目指し、草津市障害児（者）自立支援協議会の運営を通じて、 地域づくり及び関係機関との緊密化を推進します。 

主な支援内容

  • 総合的な相談支援

  • 情報提供とアドバイス

  • 関係機関との連携

  • 権利擁護

  • 受付時間・開所時間

    月曜日～金曜日 9:00～17:00

  • お問い合わせ

    077-566-5154

PLLANNING CONSULTATION

ほっとココ

ほっとココは,障害のある方やそのご家族が地域で安心して暮らせるように相談に応じ、障害福祉サービスを利用するために必要な計画を作成し、サービス事業所と連携して支援する相談支援事業所です。 「何に困っているか、不安に思うこと」「どんな毎日を送りたいか」「どんな支援があれば安心して生活できるか」などをお聞かせください。ご相談者のペースを大事にしながら一緒に考えます

主な支援内容

  • 相談支援

  • サービス等利用計画の作成

  • 関係機関との連携・調整

  • 伴走型の意思決定支援

  • 受付時間・開所時間

    月曜日～土曜日 8時30分～17時15分

    (電話での受付は8時45分～17時15分)

  • お問い合わせ

    077-569-0356

CORE CONSULTATION SUPPORT

ロコロコ

ロコロコは地域の力になれる存在をめざす障がい福祉サービス事業所です。ハワイの言葉で「島の人々」を意味する“ロコ”に思いを重ねて名づけました。誰もがほっとできる空間。利用者さんの暮らしに寄り添い、ご家族の安心も支えるサービスをお届けします。

主な支援内容

  • 創作活動（絵画、工作、手芸など）

  • 生産活動（軽作業、製品づくりなど）

  • レクリエーション活動

  • 地域交流イベントの開催

  • 日常生活の相談支援

  • 受付時間・開所時間

    月曜日～金曜日 9:00～17:15

  • お問い合わせ

    077-598-6950

SUPPORT NETWORK

草津市いきいきふれあい大運動会
草津市障害者孤立化防止事業

草津市内の障害児者がスポーツをとおして交流ができる事を主とした運動会や障害者とその家族が地域で孤立することなく生活を送っていくため、孤立を防止する活動の孤立化防止事業など草津市の委託をうけて実施しております。

主な支援内容

  • 就労に向けた支援・相談

  • 社会参加の機会づくり

  • 地域との交流活動

  • 自立生活に向けた支援

  • 関係団体との連携・協力

  • 受付時間・開所時間

    月曜日～金曜日 9:00～17:00
